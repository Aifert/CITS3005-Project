1. **Set Up the Virtual Environment**:
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    ```

2. **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

3. **Run the Application**:
    ```bash
    python run.py
    ```
    The application will be available at `http://127.0.0.1:8000`.
